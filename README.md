# clipboard

Clipboard Manager (Chrome Extension)

A Chrome extension that allows users to highlight key phrases from long readings and revisit them later in a structured, searchable clipboard history. Designed to simplify note-taking and make studying more efficient.


This extension captures highlighted text and stores it using Chrome’s local storage, creating a persistent history of copied content. Instead of losing important snippets while reading, users can easily save, organize, and reuse them when writing notes or studying.

Features
  -Save highlighted text instantly to clipboard history
  -Search through saved highlights
  -Delete individual entries or clear all history
  -Persistent storage using Chrome Extensions API
